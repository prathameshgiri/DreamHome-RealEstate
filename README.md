# DreamHome | Premium Real Estate Platform

![DreamHome Banner](https://images.unsplash.com/photo-1600596542815-ffad4c1539a9?ixlib=rb-4.0.3&auto=format&fit=crop&w=1200&q=80)

A luxurious, high-performance real estate web application designed to connect buyers and renters with premium properties. Built with a modern tech stack and featuring a sleek glassmorphism UI, this platform offers an unparalleled user experience for browsing properties, exploring top cities, and connecting with elite real estate agents.

---

## ✨ Key Features

- **Modern Luxury UI**: Fully responsive, highly polished user interface with smooth `framer-motion` animations, glassmorphism elements, and a premium color palette.
- **Comprehensive Property Listings**: Browse high-end apartments, villas, and commercial spaces with beautiful imagery and detailed specifications.
- **Agent Directory**: Discover and connect with the industry's top luxury consultants, complete with agent performance metrics and contact integration.
- **Advanced Search & Filtering**: Quickly find your dream home by location, property type, and price range.
- **Interactive Landing Page**: Engaging sections including "Explore Top Cities", "Why Choose Us", "How It Works", and "Client Testimonials".
- **Responsive Design**: Flawless experience across desktop, tablet, and mobile devices.

---

## 🛠️ Tech Stack

- **Frontend Framework**: React 18
- **Build Tool**: Vite
- **Language**: TypeScript
- **Styling**: Tailwind CSS (with custom design tokens)
- **Icons**: Lucide React
- **Animations**: Framer Motion
- **Routing**: React Router DOM

## 🚀 How to Run Locally

Follow these simple steps to get the project up and running on your local machine:

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed on your machine.

### 1. Clone the repository (if applicable)
```bash
git clone <repository-url>
cd "RealEstate Website"
```

### 2. Navigate to the frontend directory
```bash
cd frontend
```

### 3. Install dependencies
```bash
npm install
```

### 4. Start the development server
```bash
npm run dev
```

The application will now be running at `http://localhost:5173/` (or the port specified in your terminal). Open that URL in your browser to explore the website!
